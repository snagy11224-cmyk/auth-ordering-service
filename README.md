# Auth Ordering Service

Node.js e-commerce backend using Express.js, Prisma ORM, and PostgreSQL, featuring JWT authentication, modular architecture, and transactional order & inventory management.

## Features

- JWT authentication
- User authorization
- Modular Express.js architecture
- PostgreSQL database integration using Prisma ORM
- Transactional order processing
- Inventory management
- Request validation
- Centralized error handling
- RESTful API design

## Tech Stack

- Node.js
- Express.js
- PostgreSQL
- Prisma ORM
- JWT
- Git & GitHub
